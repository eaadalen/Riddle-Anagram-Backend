The purpose of this API is to be the server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Business Logic URL HTTP Method Query Parameters Request Body Data Format Response Body Data Format
Get the full list of movies /movies GET movies None A JSON object holding data about all the movies
Get data about a single movie by name /movies/[Name] GET movies, movie name None A JSON object holding data about a single movie, containing an description, genre, director, image URL, and whether it's featured or not
Get data about a genre by name /genres/[Genre] GET genres, genre name None A JSON object holding data about the genre, containing a description
Get data about a director by name /directors/[Director] GET directors, director name None A JSON object holding data about the genre, containing a bio, birth year, and death year
Allow new users to register /users/[username] POST users, username A JSON object holding data about the user, containing a username A text message indicating that the user registered successfully
Allow users to update their user info /users/[username]/info PUT users, username, info A JSON object holding data about the user, containing a username A text message indicating the username was updated
Allow users to add a movie to their list of favorites /users/[username]/favorites POST users, username, favorites A JSON object holding data about the user, containing a the movie they want to add to their favorites list A text message indicating the movie was added to their favorites list successfully
Allow users to remove a movie from their list of favorites /users/[username]/favorites/[movie_to_delete] DELETE users, username, favorites, movie to be deleted A JSON object holding data about the user, containing a the movie they want to remove from their favorites list A text message indicating the movie was removed from their favorites list successfully
Allow users to deregister /users/[username] DELETE users, username A JSON object holding data about the user, containing their username A text message that the user deregistered successfully
